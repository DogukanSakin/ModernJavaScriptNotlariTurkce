## String Karşılaştırmaları

```js
alert( 'Z' > 'A' ); // true
alert( 'Glow' > 'Glee' ); // true
alert( 'Bee' > 'Be' ); // true
```

## Diğer Türler

JavaScript, farklı türlerdeki değerleri karşılaştırırken değerleri sayılara dönüştürür.

```js
alert( '2' > 1 ); // true, 
alert( '01' == 1 ); // true

alert( true == 1 ); // true
alert( false == 0 ); // true
```
